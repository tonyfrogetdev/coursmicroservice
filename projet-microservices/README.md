# Microservice d'authentification

Ce microservice permet aux utilisateurs de s'inscrire et de se connecter avec génération de JWT.

## Routes

- `POST /api/auth/register`
- `POST /api/auth/login`

## Techno utilisées

- Express
- MongoDB Atlas
- Mongoose
- Bcrypt
- JSON Web Token
